Anime Recomendation Website

Data Gathering

or my project, I gathered data using Python's request library, Beautiful Soup (bs4), and Selenium. Specifically, I scraped a website called MyAnimeList to collect over 17K entries for my own website. The process involved sending HTTP requests to the website's server, parsing the HTML response with bs4, and using Selenium to automate interactions with the website, such as scrolling and clicking. By combining these tools and techniques, I was able to efficiently extract and process a large amount of data, which was a crucial step in building my website

Website Design

For my anime recommendation website project, I chose a Netflix-inspired design to create a clean, modern, and intuitive user interface that would be familiar and easy to use for most people. Using a combination of HTML, CSS, and JavaScript, I created a wireframe of the website layout, styled the different elements, and implemented dynamic features such as a sliding banner and search bar. Overall, the design gives the website a professional and polished look, while also showcasing anime content in a compelling way.
